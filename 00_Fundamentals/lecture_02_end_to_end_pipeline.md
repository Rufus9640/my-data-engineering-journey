# 📘 Lecture 02: End-to-End Data Pipeline

---

## 📌 What is an End-to-End Data Pipeline?

![Pipeline Overview](./00_Fundamentals/images/lecture_02/image_001.png)

An end-to-end data pipeline is the **complete flow of data** from:

> **Data Generation → Processing → Consumption**

---

## 🔁 Pipeline Stages (Core Flow)

![Pipeline Flow](./00_Fundamentals/images/lecture_02/image_002.png)

### 1. Data Generation

* Applications
* Websites
* Sensors
* User activity

---

### 2. Ingestion

* Collecting data from sources
* APIs, Databases, Event streams

---

### 3. Transformation

* Cleaning and modifying data
* Remove duplicates
* Format data
* Join datasets

---

### 4. Storage

* MySQL
* PostgreSQL
* AWS S3

---

### 5. Serving

* Dashboards
* Applications
* ML models

---

## 🎯 Final Outputs

* Analytics
* ML/AI
* Reverse ETL
